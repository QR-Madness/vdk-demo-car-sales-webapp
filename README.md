# Tech Stack

- **Backend** - Laravel
- **Database** - MySQL
- **Frontend** - Inertia.js SPA with Vue Components, Tailwind v4
- **CMS** - None; static assets

# Running the Stack

## Running Locally via Docker Compose

1. **Start the Services**

```bash
npm run services:up
```

2. **Setup Database**

```bash
npm run db:reset
```

1. **Run Application**
```bash
npm run dev
```
