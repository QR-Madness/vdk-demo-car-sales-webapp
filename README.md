# Tech Stack

- **Backend** - Laravel
- **Database** - MySQL
- **Frontend** - Inertia.js SPA with Vue Components, Tailwind v4
- **CMS** - None; static assets

# Running the Stack

## Running Locally via Docker Compose

1. **Start the Services**

```bash
npm run services:up
```

2. **Apply Database Migrations**

```bash
npm run db:migrate
```

3. **Seed Database with Content**
```bash
npm run db:seed
```

4. **Run Application**
```bash
npm run dev
```
